API Endpoints

Base URL: http://localhost:5000/api

GET /cards Fetch all cards in collection
GET /cards/:id Fetch a single card by ID
POST /cards Add a new card to collection
PUT /cards/:id Update an existing card
DELETE /cards/:id Remove a card from collection
